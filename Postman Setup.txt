Postman Setup
Base URL variable: Set {{baseUrl}} = http://localhost:8080

Auth: After login, copy the token and set {{token}} as a collection variable. For all protected requests, add header:
Authorization: Bearer {{token}}

1. Auth
POST {{baseUrl}}/api/auth/login

{
  "username": "9876543210",
  "password": "yourpassword"
}

Copy
json
Save the token from the response.

2. Users (SUPER_ADMIN only for POST/PUT/DELETE)
Method	URL	Notes
GET	/api/users	All users
GET	/api/users/{id}	By UUID
GET	/api/users/role/ADMIN	By role
POST	/api/users	Body below
PUT	/api/users/{id}	Same body
DELETE	/api/users/{id}	No body
POST body:

{
  "name": "John Doe",
  "mobile": "9876543210",
  "email": "john@example.com",
  "password": "pass123",
  "role": "ADMIN"
}

json

Tip: Create a Postman Collection with a "Login" request that has a test script to auto-save the token:

// In Tests tab of login request
pm.collectionVariables.set("token", pm.response.json().token);

Copy
js
This way every subsequent request automatically uses the latest token
